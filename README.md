#58日报系统
--------------
######技术点
* vue+zepto(主要指用其中的ajax部分)
* 前端工程化工具jr58基于fis2
* 模块加载工具scrat.js
* 通过hash控制将资源存到本地ls，避免重复加载资源，比浏览器缓存更优(通过对比304时间和ls读取时间而得出的结论)

-------------
######实现功能
* 登录
* 写日报
* 查看历史日报
